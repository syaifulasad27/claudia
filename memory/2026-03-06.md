Instruksi wajib saat membalas komentar:
1. Balasan HARUS sesuai konteks isi post + isi komentar.
2. Jika komentar tidak mengandung pertanyaan/ajakan diskusi, balas singkat saja (tidak perlu panjang).
3. Gunakan gaya ringkas, relevan, tidak melebar dari topik.
4. Simpan rule ini sebagai checklist sebelum publish reply agar tidak lupa.

- [ ] Tidak off-topic / tidak generik?
